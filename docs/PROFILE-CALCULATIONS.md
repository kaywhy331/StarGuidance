# Profile calculations

## Implemented

Pythagorean numerology `pythagorean-v1` deterministically calculates Life Path, Expression, Soul Urge, Personality, and Birthday values. Master numbers 11, 22, and 33 are preserved. Spaces, hyphens, apostrophes, and Latin diacritics are normalized in a separately recorded rendering; the encrypted original name is never overwritten. Non-Latin names require a user-confirmed Latin rendering and are never silently transliterated.

Dreamspell `dreamspell-anchor-1987-07-26-kin34-v1` produces Kin, tone, solar seal, color, and algorithm version from the Gregorian date, anchored to 26 July 1987 as Kin 34. It is labeled `implemented_pending_approved_reference_dataset`; production certification requires the approved decoder dataset and terminology/rights review.

Approximate birth times remain start/end ranges. They are never collapsed to a midpoint or described as exact. Birth time requires a birthplace or authoritative timezone. Unknown time remains null.

## Typed unavailable systems

Western astrology, Whole Sign houses, Placidus houses, and BaZi Four Pillars return typed unavailable results. Activation requires compatible licensing, explicit conventions, golden-reference boundary datasets, and domain-expert approval. No placeholder chart fact is returned.
